Research

  • Emission Measure Studies in Hydrodynamic Models
    • The nanoflare heating theory of the solar corona has garnered much attention since its conception by Parker (1988). However, emission signatures from nanoflare heating events are extremely difficult to detect. One solution is to use hydrodynamic models to calculate observables from nanoflare-heated coronal loops and compare to observations. I have adapted the popular enthalpy-based thermal evolution of loops (EBTEL) model by Klimchuk et al. (2008), Cargill et al. (2012a), and Cargill et al. (2012b) to treat the evolution of the electron and ion temperatures and pressures separately while maintaining the assumption of quasi-neutrality. This model has been successfully benchamarked against the more robust 1D hydrodynamic code HYDRAD. Our group is interested in using this two-fluid EBTEL model to explore large parameter spaces not accessible to slower hydrodynamic models in order to better constrain nanoflare heating properties.

  • Automatic Event Detection in Active Region Cores

    • Sample composite time series with five overlapping gaussian pulses.

    Resulting sources after separation by NMF algorithm; even the most strongly overlapping sources (panels 3 and 4) are recovered as distinct pulses.

    I am interested in using non-negative matrix factorization (NMF), a matrix deconvolution technique with a wide variety of applications, to automatically identify emission signatures of nanoflare-heated plasma. For a given number of source functions of arbitrary shape, NMF minimizes a cost function in order to fit the sum of these individual sources to the convoluted truth function. In this case, the "truth" is the composite intensity time-series and the sources are the responses to individual nanoflare events. Identifying the correct number of source functions and their associated amplitudes will allow for improved constraints on both nanoflare heating frequency and strength. Before such an algorithm can be applied to observational data, its performance must be assessed through application to forward-modeled results. Currently, our group is working toward an NMF algorithm capable of accurately reproducing the nanoflare heating frequency of forward-modeled nanoflare-heated plasmas from hydrodynamic codes.

Software

  • EBTEL
    The Solar Physics Research Group at Rice University maintains the enthalpy-based thermal evolution of loops (EBTEL) model by Klimchuk et al. (2008), Cargill et al. (2012a), and Cargill et al. (2012b), written in IDL. The source code and documentation can be found on the GitHub page.
  • EBTEL-C
    In addition to the original EBTEL model, our group maintains a translation of the EBTEL model into the C programming language. This version allows for much quicker run times (e.g. approximately 10 s for an 80,000 s run) and includes a fourth-order Runge-Kutta solver. The model also uses an easily configurable XML file to manage the input parameters. EBTEL-C has been carefully benchmarked against the original EBTEL code. The source code and documentation, as well as several Python scripts to help configure and run the model, can be found on the GitHub page
  • SolarNMF
    Recently, I have been adapting some common non-negative matrix factorization (NMF) algorithms for use with solar data, either in a time series or image format. Currently, we are testing this software against forward modeled data from both HYDRAD and EBTEL. The Python source code is available on GitHub. If you are interested in contributing to this project, please contact me.
